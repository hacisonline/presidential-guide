# Tesseract Home — User Guide · The Presidential

Interactive, customer-facing user guide for the Tesseract Home deployment at
The Presidential (Bandra West, Mumbai). A single self-contained page —
`index.html` — in the Tesseract brand, covering the resident app, the building
team's desks and portal, and the local-inference intelligence pipeline.

Deployed with GitHub Pages from the repo root. No build step.
